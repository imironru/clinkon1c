# Clinkon1C

> Утилита администратора для очистки артефактов 1С на Windows и RDP-серверах.

![Platform](https://img.shields.io/badge/platform-Windows-blue)
![.NET](https://img.shields.io/badge/.NET-8.0_LTS-purple)

## Описание

Clinkon1C — консольная утилита с псевдографическим интерфейсом (TUI) для администраторов систем 1С. Позволяет просматривать, анализировать и очищать:

- **Кэш** клиентских приложений 1С (по пользователям и базам)
- **Логи** платформы и сервера 1С, журналы регистрации _(Phase 2)_
- **Шаблоны** конфигураций 1С _(Phase 2)_

> ⚠️ **Утилита предназначена исключительно для администраторов систем.**  
> Неправильное использование может привести к потере данных пользователей 1С.

## Возможности

- Навигация по дереву как в файловом менеджере
- Два вида дерева: по пользователю и по базе (переключение Tab)
- Множественное выделение с подсчётом объёма
- Режим Dry Run — предпросмотр перед удалением
- Защита критических файлов (`*.lic`, `*.pfl`, `*.1CD` и др.)
- Определение блокирующих процессов через Windows Restart Manager API
- Лог всех операций в `C:\ProgramData\Clinkon1C\logs\`
- Опциональный Backup перед удалением
- Исключения по пользователям и базам
- Автообновление с GitHub

## Требования

- Windows 10 / Windows Server 2012 R2 и выше
- 64-bit
- Права администратора
- .NET не требуется (self-contained exe)

## Установка

1. Скачайте `Clinkon1C.exe` из [Releases](../../releases/latest)
2. Положите в удобное место (например `C:\Tools\`)
3. Запустите от имени администратора

## Разработка

```bash
git clone https://github.com/*/Clinkon1C.git
cd Clinkon1C
dotnet build
```

Сборка релизного exe:

```bash
dotnet publish -r win-x64 --self-contained true -p:PublishSingleFile=true -c Release
```
