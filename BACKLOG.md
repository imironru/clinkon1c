# Backlog

Задачи для следующих версий. Закрытые задачи — в [CHANGELOG.md](CHANGELOG.md).

---

## Открытые задачи

### Правила брандмауэра Windows для 1С
Модуль Диагностики → статус и управление правилом.

Наработки: `inbox/register-ras.zip` → `1c-firewall.bat`.

Порты: 1540 (агент), 1541 (менеджер), 1545 (RAS), 1550, 1560–1591 (рабочие процессы).

- Показывать: есть ли правило «1c», какие порты открыты
- Действия: создать/обновить правило, удалить правило
- Реализация: COM `HNetCfg.FwMgr` / `INetFwRules` (без `netsh`)

---

### Редактор inetcfg.xml (прокси-настройки платформы)
Раздел Конфиги платформы. Сейчас `CanEdit=false`.

3 поля: адрес прокси, порт, исключения.

---

### DT backup / restore
Выгрузка и загрузка файловой базы через `1cv8.exe /DumpIB` и `/RestoreIB`.

- В разделе Базы: действие «Выгрузить в DT» / «Загрузить из DT»
- Показывать прогресс через `ShowProgress`

---

### Редактор ClientUpdate.cfg
Управление авто-обновлением клиента 1С.

- Включить / отключить авто-обновление
- Прописать URL дистрибутива

---

### Редактор 1cestart.cfg
Настройки запуска по умолчанию (версия платформы, параметры).

---

### Windows EventLog — события 1С
Просмотр событий 1С из журнала Windows Application Log.

- Источники: `1C:Enterprise`, `1CV8`
- Показывать через `ShowText` со скроллом

---

### Drill-down в сводке (правая панель)
Из правой панели (Диагностика) нажать Enter на строке → открыть соответствующий модуль.

Например: Enter на строке версии 1С → перейти в раздел Версии.
